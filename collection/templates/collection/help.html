网站帮助

一、帐号

  1. 创建帐号需要联系管理员，提供邮箱。创建后密码为随机值，需要自行重置密码。

二、标注软件

  1. 提供 源代码 和 Windows 可执行程序。

  2. 从 Github Release 下载最新版本及使用帮助，或者直接从下面的链接下载v0.0.3版本。

    • imageviewer.zip
    • help.pdf

三、获取图片包

  1. 在 Packages 页面点击“Ask for more packages”分配一个新的图片任务包。
  2. 把图片包下载到本地。
  3. 单个图片包解压到单个文件夹内。不要拆分成多个文件夹，也不要把多个图片包合并到同一个文件夹。

四、标注任务

  1. 根据“标注需求”和标注软件的帮助文档完成标注。
  2. 用标注软件导出标注，在“Upload Annotations”上传。
  3. 上传后可以获取下一个图片包。

五、交叉验证

  1. 当另一个领取相同图片包的用户上传了标注时，双方都会得到Feedbacks。
  2. 在图片包详情页面下载Feedbacks，用标注软件导入它。
  3. 检查Feedbacks里所有错标、漏标、多标的地方，修改后重新导出、上传。
