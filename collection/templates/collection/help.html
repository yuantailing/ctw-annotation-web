网站帮助

一、帐号

  1. 创建帐号需要联系管理员，这个网站没有注册页面。

二、标注软件

  1. 提供 源代码 和 Windows 可执行程序。

  2. 从 Github Release 下载最新版本及使用帮助，或者直接从下面的链接下载v0.0.5版本。

    • imageviewer.zip
    • help.pdf

三、获取图片包

  1. 在 Packages 页面点击“Ask for more packages”分配一个新的图片任务包。
    注：你获取的第1个图片包是示例图片包，只含3张图片，用于让你熟悉标注软件，上传标注，交叉验证的流程。
  2. 把图片包下载到本地。
  3. 单个图片包解压到单个文件夹内。不要拆分成多个文件夹，也不要把多个图片包合并到同一个文件夹。
    Tips：同一个包里的图片的拍摄位置有一定连续性，这是为了方便你根据上下文更迅速地判断出文字。方向（Direction）0表示拍摄角度相对于摄影车前进方向是向后的，1向左，2向前，3向右。

四、标注任务

  1. 根据“标注需求”和标注软件的帮助文档完成标注。
  2. 用标注软件导出标注，在“Upload Annotations”上传。
  3. 上传后可以获取下一个图片包。

Mask 标记需求

把所有没有标注出来的汉字标记为 Mask（无法辨别）。

  1. 一组字只需要一个标记，不要单独标记每一个字。
    正确：
    错误：
  2. 高度不同的几组字要分开标记。
    （红色、白色、黑色的汉字分开标记，一共三组标记）
  3. 不标记被大面积遮挡的字、残缺的字。但是如果它们跟需要标记的字是同一组字，那么也可以标记出来，不用特意避开。
    不标：
  4. 不标记数字、字母。但是如果它们跟需要标记的字是同一组字，那么也可以标记出来，不用特意避开。
  5. 不能分辨是否是汉字也可以标记。
  6. 要标记以下内容：
    很小的字：（不标记数字）
    车牌：
    因全景拼接错误没标注出来的字：
    没有标注出来的艺术字：
    因模糊没有标出来的字：
    因其它原因没有标注出来的字：
  7. 不标记太小的字。
    （不标记右边的“元宝？”，要标记左边的“元宝？”，不标记左边“元宝？”两边的小字）
  8. 如果标注的一组字漏掉了几个字，不要标记为 Mask，而是重新标注这组字。

属性标注需求

  1. 为每个已被标注的字（含数字、字母）勾选属性。
  2. 标注任意一项属性都会自动勾选“标记为‘已经标注属性’”。如果没有任何属性需要勾选，也要勾选这一项，这是为了与未浏览的字区分开。
  3. 不用标注 Mask 的属性。
  4. 倾斜示例（倾斜指拍照时字没有正对镜头。不是指这种斜着印刷的字）
  5. 被遮挡示例（只有“养”“殖”勾选“被遮挡”属性，“湘”“湖”“水”“产”“场”不勾选）
  6. 立体字示例
  7. 背景复杂示例
